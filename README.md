# FCUCourseBot

## Introduction
聽說逢甲的課有夠難選，所以就幫忙寫一個。

## Depndencies
* requests
* bs4
* ddddocr

## Usage 

1. 下載到電腦

```
git clone https://github.com/mhy1264/FCUCourseBot.git
```

2. 安狀套件 
```
pip install -r requirements.txt
```

3. 新增 `accounts.ini`，格式如下
```
[Default]
Account=
Password=
```
4. 在選課系統把想要搶的課放在關注清單

5. 執行
```
python main.py
```

> **Warning** <br>
> 請斟酌使用本機器人程式，並自行負責使用後所造成的損失!
